/* 应用范围:  */
---------------
	用于需要显示新闻及新闻详细信息的页面。


/* 文件说明:  */
---------------
/*********************************************************
******			文件: Info.ascx						******
*********************************************************/
作用:
	按新闻类别显示所有的新闻列表。
	列表显示信息：新闻标题、发布日期、阅读次数。
	点击“更多”链接，跳转到该类别新闻的所有新闻列表页面。
	

/*********************************************************
******			文件: InfoDtl.ascx					******
*********************************************************/
作用:
	显示某个新闻的详细内容，同时可以打印该新闻内容。
	相关信息栏显示的是当前新闻所属类别中的其它相关新闻，
	点击某一新闻标题链接，即可查看相应新闻的详细内容。
	

/*********************************************************
******			文件: InfoMore.ascx				******
*********************************************************/
作用:
	在所有新闻列表中，点击某个新闻类别的“更多”链接，即跳转到该控件页面。
	显示该类别新闻的所有新闻列表。
	

/* 调用接口或方法:  */
----------------------
直接将所需控件拖拽到页面的某个位置即可。
要求：
	InfoDtl.ascx控件所在页面命名：InfoDtl.aspx，
	InfoMore.ascx控件所在页面命名：InfoMore.aspx
