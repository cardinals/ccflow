
ccflow 插件扩展应用说明:


以目录下的组件层是ccflow的扩展应用，您可以根据自己的项目性质增加响应的插件.
----------------------------------------------------------------------------------



+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++
工程类型流程项目插件:

流程特征:
1, 每个流程都与项目有关系。
2, 项目有项目组，项目成员.
3, 项目的流程的数据结算到该项目上.
4, 一个项目有一序列的工作岗位, 比如项目经理，高级工程师, 初级工程师. 这些岗位存储在 Port_Station 表里.
5, 项目的权限管理以项目以单位，
   比如：一个人员(Port_Emp) 在一个项目中是项目经理，在另外一个项目中是高级工程师.


提供的功能:
1, 项目维护，项目成员维护.
2, 项目流程查询.
3, 项目资料树, 设计该项目的所有资料文档都挂在此资料树上。

特殊约定:
1, 设计项目的流程每个节点表单必须有PrjNo,PrjName 两个字段，ccflow 会依据这两个字段来确定是不是项目流程并进行特殊处理.
