/* 应用范围:  */
---------------
	用于需要显示软件下载列表及软件详细信息的页面。


/* 文件说明:  */
---------------
/*********************************************************
******			文件: Soft.ascx						******
*********************************************************/
作用:
	按软件类别显示所有的软件列表。
	列表显示信息：软件名称、上传日期、下载次数，并提供下载链接。
	点击“更多”链接，跳转到该类别软件的所有软件列表页面。
	

/*********************************************************
******			文件: SoftDtl.ascx					******
*********************************************************/
作用:
	显示某个软件的详细信息，同时可以下载该软件。
	相关信息栏显示的是当前软件所属类别中的其它相关软件，
	点击软件名称链接，即可查看相应软件的详细信息。
	页面跳转需要传递的参数：RefNo = Soft对象的No值 。
	

/*********************************************************
******			文件: SoftMore.ascx				******
*********************************************************/
作用:
	在所有软件列表中，点击某个软件类别的“更多”链接，即跳转到该控件页面。
	显示该类别软件的所有软件列表。
	页面跳转需要传递的参数：RefNo = SoftSort对象的No值 。
	

/* 调用接口或方法:  */
----------------------
直接将所需控件拖拽到页面的某个位置即可。
要求：
	SoftDtl.ascx控件所在页面命名：SoftDtl.aspx，
	SoftMore.ascx控件所在页面命名：SoftMore.aspx
